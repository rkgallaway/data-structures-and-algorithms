# Insert and shift middle index of array
Code Challange 02: Reverse an array 
paired with Jared Pattison


## Challenge
Write a function called insertShiftArray which takes in an array and the value to be added. Without utilizing any of the built-in methods available to your language, return an array with the new value added at the middle index.


## Approach & Efficiency
We declared a function that iterated thru an array, inserted the same value into new array for the first half, inserted the given value at the midpoint, and added the rest of the original array after the given value insertion.
